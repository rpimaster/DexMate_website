DexMate Setup Guide

Get started with DexMate in minutes with our step-by-step guides

Windows Setup Guide

1

Download the Installer

Get the DexMate installer for Windows from our download page.

If your browser blocks the download:

  • Chrome: Click "Keep" in the download warning
  • Edge: Click "Keep anyway" in the prompt
2

Run the Installer

Double-click the downloaded .exe file and follow these steps:

1

If you see "Windows protected your PC", click More info

2

Click Run anyway

3

Follow the on-screen instructions

3

Launch and Log In

After installation, DexMate will launch automatically and prompt you to configure your data source. You can choose between Dexcom and Nightscout:

  • Dexcom: Enter your Dexcom username, password, and select your region (US, OUS, or Japan).
  • Nightscout: Provide your Nightscout URL and API Secret if required.

Once your preferred source is set up, click Save Configuration to begin monitoring your glucose data.

Optional: Run on Startup

To launch DexMate automatically when you start your computer:

1

Press Win + R, type shell:startup

2

Copy the DexMate shortcut to this folder

macOS Setup Guide

1

Download the Installer

Download the DexMate.dmg file from our download page.

If your browser blocks the download, click "Keep" in the warning message

2

Install the Application

Double-click the downloaded .dmg file and drag DexMate.app to your Applications folder.

3

Launch the Application

Open your Applications folder and double-click DexMate.app. If you see a security warning:

Allowing DexMate to Run

If macOS prevents DexMate from opening, use this method:

1
System Settings Method
  1. Go to System Settings > Privacy & Security
  2. Scroll to the Security section
  3. Find the message: "DexMate was blocked..."
  4. Click Allow Anyway
  5. Try opening DexMate again

DexMate will now launch and prompt you to configure your data source. You can choose between Dexcom and Nightscout:

  • Dexcom: Enter your Dexcom username, password, and select your region (US, OUS, or Japan).
  • Nightscout: Provide your Nightscout URL and API Secret if required.

Once your preferred source is set up, click Save Configuration to begin monitoring your glucose data.

Linux Setup Guide

1

Install via Snap

Open your terminal and run:

sudo snap install dexmate
2

Launch DexMate

You can find DexMate in your application menu or run:

dexmate
3

Log In

Enter your Dexcom credentials when prompted.

Automatic Updates

Snap installations update automatically. To manually update:

sudo snap refresh dexmate
1

Download the ZIP

Download the Linux ZIP file from our download page.

2

Extract Files

Right-click the downloaded ZIP file and select "Extract Here".

3

Make Executable

Right-click the DexMate file → Properties → Permissions → Check "Allow executing file as program".

4

Launch and Log In

After installation, double-click the DexMate file to launch it. DexMate will launch and prompt you to configure your data source. You can choose between Dexcom and Nightscout:

  • Dexcom: Enter your Dexcom username, password, and select your region (US, OUS, or Japan).
  • Nightscout: Provide your Nightscout URL and API Secret if required.

Once your preferred source is set up, click Save Configuration to begin monitoring your glucose data.

Optional: Run on Startup

To launch on login:

  1. Install Startup Applications: 
    sudo apt install gnome-startup-applications
  2. Open Startup Applications Preferences
  3. Click Add, name it "DexMate"
  4. Browse to the DexMate file
  5. Click Add
