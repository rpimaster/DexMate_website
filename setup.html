Setup Guide for DexMate

Follow these simple steps to get DexMate up and running on your device.

Setup Guide for Windows

Step 1: Download the installation file

Select your location, measurement units, and operating system from the download page and download the appropriate installation file for your device.

If your browser blocks or questions the download, please follow these steps:

  • Google Chrome: Look for a message at the top-right corner of the browser window. If it says the download was blocked, click "Download" to continue downloading the file.
  • Microsoft Edge: Check the top-right corner of your browser window for a download prompt. Click "More" (three dots) and then "Keep" to proceed with the download. You might be asked again if you trust the page. Click "Show more" and then "Keep anyway". The file should now download successfully.

For additional help, visit our Support Page.

Step 2: Run installation file

Once the download is complete, open the file explorer and go to the directory where you downloaded the installation file. Then double-click on it. It's possible that you will get this blue window stating "Windows protected your PC". In this situation, just click more info and then run anyway. Now go through the installation setup (follow the on-screen instructions).

Step 3: Run the app

When the installation has finished, the app should open automatically. If not, just open it like any other app. Now you need to enter your Dexcom credentials and hit the log-in button. Enjoy easier life with glucose monitoring on your machine!

*Optional Step: Run on Startup

If you want DexMate to run automatically when you log in to your PC, follow these steps:

Click the Windows key + R and type “shell:startup”. It will open a folder. Now you should copy the DexMate shortcut from your desktop or wherever you have saved it to this folder. Now the app will start automatically when you log in to your PC.

Setup Guide for MacOS

Step 1: Download the installation file

Select your location, measurement units, and operating system from the download page and download the appropriate installation file for your device.

If your browser blocks or questions the download, please follow these steps:

  • Google Chrome: Look for a message at the top-right corner of the browser window. If it says the download was blocked, click "Download" to continue downloading the file.
  • Microsoft Edge: Check the top-right corner of your browser window for a download prompt. Click "More" (three dots) and then "Keep" to proceed with the download. You might be asked again if you trust the page. Click "Show more" and then "Keep anyway". The file should now download successfully.

For additional help, visit our Support Page.

Step 2: Unzip the folder

Once the download is complete, open Finder and go to the directory where you downloaded the zip file. Then double-click on it to extract the contents. You should see the DexMate.app file along with the license.

Step 3: Run the app

After unzipping the folder, locate DexMate.app (you can see just "DexMate"). Drag and drop the DexMate.app file into your Applications folder for easy access.

Step 4: Launch the App

Open the Terminal application on your Mac. Then, paste the following commands one by one into the Terminal:

  • sudo chown -R $USER /Applications/DexMate_mmol/DexMate.app
  • sudo chmod -R 755 /Applications/DexMate_mmol/DexMate.app
  • sudo xattr -rd com.apple.quarantine /Applications/DexMate_mmol/DexMate.app

Note: Adjust the file path based on your version of the DexMate app. For example, replace /Applications/DexMate_mmol/DexMate.app with /Applications/DexMate_mgdl_us/DexMate.app if needed.

If prompted for your password, type it in and press Enter. This ensures the security of the changes.

Important: Please double-check the commands before running them. We do not take any responsibility for issues caused by incorrectly typed commands or unintended changes to your system.

Once you've completed these steps, you can close the Terminal window.

Launching DexMate

To run DexMate:

  • Open the Applications folder.
  • Locate DexMate.app and double-click it to launch.

If you receive a warning about the app being from an "unidentified developer," follow these steps:

  1. Right-click (or Control-click) on DexMate.app.
  2. Select Open from the context menu.
  3. In the warning message that appears, click Open again to confirm.

After confirming, the app will launch normally, and you won’t see this warning again.

Enjoy using DexMate!

Setup Guide for Linux

Step 1: Download the installation file

Select your location, measurement units, and operating system from the download page and download the appropriate installation file for your device.

If your browser blocks or questions the download, please follow these steps:

  • Google Chrome: Look for a message at the top-right corner of the browser window. If it says the download was blocked, click "Download" to continue downloading the file.
  • Microsoft Edge: Check the top-right corner of your browser window for a download prompt. Click "More" (three dots) and then "Keep" to proceed with the download. You might be asked again if you trust the page. Click "Show more" and then "Keep anyway". The file should now download successfully.

For additional help, visit our Support Page.

Step 2: Unzip the folder

Once the download is complete, open your file manager and go to the directory where you downloaded the zip. Then right-click on it and select "Extract Here".

Step 3: Run the app

After unzipping the folder, open the extracted DexMate folder and locate the DexMate file. Right-click on the file and select Properties. In the Properties window, navigate to the Permissions tab. Ensure that all permissions are set to Read and Write and that the option to Allow executing file as a program is enabled.

Once you've made these changes, click Save (if required) or simply close the Properties window. You can now double-click on the DexMate file to run the application.

*Optional Step: Run on Startup

If you want DexMate to run automatically when you log in to your PC, follow these steps:

Open terminal and run this command: sudo apt install gnome-startup-applications

Then find the startup application preferences app and open it.

Click the “Add” button and enter a desired name. In the command section, select "Browse...," navigate to the unzipped folder, and choose the DexMate file. Optionally, add a comment, then click “Add” to complete the process.

Now you can close the application. The app should run automatically when started and logged in to the PC.
